# Used Car Data Cleaning & Baseline Model

## Objective
To clean a messy used car dataset and compute a baseline MAE model.

## Tasks Performed

### Task 1: Data Exploration
- Identified missing values
- Found incorrect data types
- Detected duplicates and inconsistent text formatting

### Task 2: Data Cleaning
- Removed rows with missing target values
- Imputed missing data using median/mode
- Standardized text columns
- Extracted numeric values from mileage
- Removed duplicate rows

### Task 3: Baseline Model
- Used mean selling price as prediction
- Evaluated using Mean Absolute Error (MAE)
